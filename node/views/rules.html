Next Code Golf Rules

There will be ten problems. The competition will last 60 minutes. Contestants need to bring their own laptops.

Submit solutions to the problems in Python. The solutions will be tested on various judge inputs. Correct solutions will be scored by the number of non-whitespace characters in the solution. In order to deter from one-letter variable names and encourage the use of library functions, all identifiers will count as only one character each. Obviously, the lowest score wins.

Specifically, we remove all whitespace (spaces, tabs, newlines) and collapse all contiguous substrings of alphanumeric characters (A-Z, a-z, and underscore) into one character, unless they are in strings. For example:

  • print 'hi': 5 (one for print, and four for the string 'hi')
  • s.replace(' ', ''): 11 (the space after the comma is not counted, but the space in the string is)

Important: file operations, network operations, and eval() functions (as well as other "bad" functions) are NOT allowed.

For each problem, the top scorers are awarded "global points" in Mario Kart style. The person with rank r is given n points. Ties are broken by time submitted.
